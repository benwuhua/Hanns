#### 2.2 优化历程 (OPT-005)

| 版本 | 优化内容 | 效果 |
|-----|---------|------|
| v1 | Rayon 并行搜索（par_iter） | +75x |
| v2 | select_nth_unstable + l2_distance_sq | +41% |
| v3 | HashMap → Vec + 批量 SIMD | +52% |
| v4 | l2_batch_4 批量处理 | +104% |
| **v5 (最终)** | 扁平化重构 + TopKAccumulator | **+335%** ✅ |

**核心改动** (src/faiss/ivf_flat.rs):
- 扁平化存储布局（Vec<Vec<_>> → Vec<_>）
- 两阶段 add() 方法（批量构建连续倒排表）
- TopKAccumulator 固定容量堆（避免候选全量收集）
- scan_cluster() 优化（保留 SIMD batch_4）

**性能提升**：
- 10K: QPS 4,038 → 12,775 (+216%)
- 50K: QPS 922 → 4,054 (+339%)
- 100K: QPS 470 → 2,045 (+335%)

   - Random 数据: 真正伪随机，均匀分布
2. **召回率对比**:
   - Pattern: R@10 = 74-78% ❌
   - Random: R@10 = 100% ✅
3. **结论**: Random 数据反映真实性能
